# QC_composer
박성욱, 정휘온, 최승현 조입니다.

====
## Tutorials

이 프로젝트는 **직접 회로를 구성하며 양자 개념을 이해하는 참여형 튜토리얼**을 제공합니다. 각 튜토리얼은 단계별 목표, 즉시 피드백, 힌트/리셋 기능을 통해 학습자가 스스로 개념을 검증하도록 설계되었습니다.

### 제공 튜토리얼

* **Hadamard Gate**
  *학습 목표*:

  * 계산 기저 |0⟩, |1⟩와 중첩 상태의 차이를 구분한다.
  * Hadamard 게이트가 상태 벡터에 미치는 수학적·물리적 의미를 이해한다.
  * 측정 분포가 50:50으로 나타나는 이유를 회로 수준에서 설명할 수 있다.

* **Controlled Gates (CNOT)**
  *학습 목표*:

  * 제어 큐비트와 타깃 큐비트의 역할을 명확히 구분한다.
  * 단일 큐비트 연산과 제어 연산의 본질적 차이를 이해한다.
  * CNOT을 통해 생성되는 상관관계와 얽힘 상태를 구분해 설명할 수 있다.

* **Quantum Fourier Transform (QFT)**
  *학습 목표*:

  * 위상(phase)이 양자 상태에서 갖는 정보적 의미를 이해한다.
  * Hadamard 및 제어 회전 게이트가 위상 누적을 구현하는 방식을 설명한다.
  * QFT가 위상 정보를 계산 기저 확률로 변환하는 과정을 단계별로 추적한다.

* **Superdense Coding**
  *학습 목표*:

  * 얽힘이 고전적 정보 전달 한계를 어떻게 확장하는지 이해한다.
  * 송신자(Alice)와 수신자(Bob)의 연산을 역할별로 구분해 설명한다.
  * 전체 프로토콜을 하나의 회로 흐름으로 재구성할 수 있다.

* **Deutsch–Jozsa Algorithm**
  *학습 목표*:

  * 고전적 알고리즘과 양자 알고리즘의 질적 차이를 문제 설정 차원에서 이해한다.
  * 오라클(oracle)이 양자 회로에서 어떤 역할을 하는지 설명할 수 있다.
  * 단 한 번의 오라클 호출로 함수가 상수(constant)인지 균형(balanced)인지 판별하는 원리를 이해한다.
  * 간섭(interference)이 계산 결과를 증폭·소거하는 과정을 회로 수준에서 추적한다.

### 학습 목표

이 튜토리얼 시스템의 목표는 단순한 회로 암기가 아니라, **양자 회로가 만들어내는 상태 변화의 원리를 스스로 구성·검증하며 이해하는 것**입니다.

학습자는 각 튜토리얼을 통해 다음을 달성하게 됩니다:

* 단일 큐비트 및 다중 큐비트 상태 변화의 물리적 의미 이해
* 게이트 조합이 측정 확률과 위상에 미치는 영향 분석
* 제어 게이트를 통한 상관관계와 얽힘의 형성 과정 이해
* 프로토콜 수준에서 양자 알고리즘의 정보 흐름 파악
* 추상화된 회로 표현(IR)과 실제 실행 회로(Qiskit) 간의 대응 관계 이해

### 세션 구성

각 튜토리얼은 동일한 구조의 학습 세션으로 구성되어 있어, 개념 간 비교와 확장이 용이합니다.

1. **이론 소개**
   해당 튜토리얼에서 다루는 핵심 개념과 물리적 의미를 간략히 설명합니다.

2. **단계별 미션 (Steps)**
   학습자가 직접 회로를 구성해야 하는 명확한 목표가 제시됩니다. 각 단계는 하나의 개념에만 집중하도록 설계되었습니다.

3. **즉시 검증 (Check)**
   현재 회로 상태가 목표 조건을 만족하는지 확인합니다. 결과는 성공/실패뿐 아니라 관찰 포인트 중심으로 피드백됩니다.

4. **힌트와 리셋 (Hint / Reset)**
   시행착오를 허용하되, 개념적 방향을 잃지 않도록 최소한의 가이드를 제공합니다.

5. **결과 해석**
   측정 분포와 회로 구조를 함께 해석하며, 이전 단계와의 차이를 비교합니다.

### 특징

* 드래그 앤 드롭 회로 편집
* 단계별 체크(Check)와 힌트(Hint)
* 튜토리얼 전용 독립 회로 상태 관리
* Qiskit Export 지원 (학습용 IR → 실제 회로 변환)

> 튜토리얼은 학습 목적의 **추상화된 내부 표현(IR)** 을 사용하며, 실제 실행은 Qiskit 회로로 변환됩니다.
